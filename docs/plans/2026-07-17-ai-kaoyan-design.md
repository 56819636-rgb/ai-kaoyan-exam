# AI考研轻量答题网站设计

## 产品与界面

网站服务于个人的 155 天 MBA 备考实验，核心是“打开就能做题，退出也不丢”。视觉采用安静克制的纸质考场风格：暖米色背景、深墨绿色主色、朱砂色提醒，避免多余动效。390px 手机是主画布，答题页顶部只保留题号与计时，底部固定操作栏；桌面端限制内容宽度，避免长行难读。

## 架构与数据流

使用 React + TypeScript + Vite 的纯前端单页应用，HashRouter 避免 GitHub Pages 子路径刷新问题。内置试卷由 `public/exams/index.json` 枚举，启动时 fetch 并缓存；本地导入试卷经过运行时校验后写入 localStorage。所有持久化 key 带统一版本前缀，分别保存导入试卷、进行中进度、历史成绩、错因和设置。考试进度每次答题、标记、跳题及计时都会保存，恢复时按 examId 找回。

## 评分、错误与测试

正式自动评分仅处理 `single_choice`，其他预留题型只保存作答并标记为不可自动评分。结果保存试卷快照，确保删除导入卷后历史仍可查看。校验错误返回精确到字段路径的中文提示；存储解析失败时使用安全默认值，不让页面崩溃。单元测试覆盖评分、模块/知识点/难度统计以及进度序列化恢复；最终执行测试、TypeScript 构建和 PWA 产物检查。
